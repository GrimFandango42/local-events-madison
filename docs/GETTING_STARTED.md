# Getting Started

This quick guide helps you run the Local Events platform locally.

- Requirements: Node.js 18+, npm, SQLite (bundled), prisma
- Steps:
  - npm install
  - cp .env.example .env.local (or set env vars)
  - npm run db:generate
  - npm run db:push
  - npm run db:seed
  - npm run dev

Visit http://localhost:3000.
